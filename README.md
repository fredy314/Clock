# Smart Clock (ESP32-C3 + MAX7219 + DS3231 + DHT22)

Розумний годинник на базі ESP32-C3 з відображенням часу, температури та вологості на LED-матриці 8x32.

## Основні можливості

- **Циклічне відображення**: 5 секунд показує час (HH:MM), 2 секунди — температуру та вологість.
- **Синхронізація часу**: 
    - Пріоритетно через Wi-Fi (NTP).
    - Автономно через модуль RTC (DS3231) з механізмом повторних спроб при збої зчитування.
    - Ручна синхронізація через USB/Serial за допомогою Python-скрипта.
    - ESP-NOW синхронізація (маякування часу іншим пристроям).
- **Сенсори**: Вимірювання температури та вологості за допомогою датчика DHT22 (AM2302).
- **MQTT & Home Assistant**: 
    - Автоматичне виявлення пристрою (Discovery).
    - Передача температури, вологості та заряду батареї кожну хвилину.
    - Округлення значень до 1 знака після коми.
- **Моніторинг батареї**:
    - Асинхронне вимірювання у фоновому таску FreeRTOS.
    - Дворівневе згладжування: середнє за 5 замірів + медіанний фільтр за 10 секунд для максимальної стабільності.
    - Динамічна індикація заряду на матриці (тільки в режимі годинника).
- **OTA Update**:
    - Оновлення прошивки та файлів SPIFFS через веб-інтерфейс.
    - Перевірка сумісності (назва проекту) та захист від випадкового даунгрейду.
    - Відображення поточної версії та назви проекту на сторінці оновлення.
- **Діагностика**:
    - Збір та перегляд логів за останні 5 секунд через веб-інтерфейс (клік на копірайт).
    - Таймаут AJAX-запитів (5с) для стабільної роботи в нестабільних мережах.
    - Швидкий перегляд температури/вологості на LED-матриці при кліку на відповідну картку в UI (на 2 секунди).
- **Дизайн**: 
    - Кастомний шрифт 5x8 (на всю висоту матриці 8 пікселів) та система автоматичного центрування.
    - Плавний хід годинника в браузері (оновлення щосекунди локально).

## Схема підключення (GPIO)

| Компонент | Пін | Опис |
|-----------|-----|------|
| **MAX7219** | DIN: 7, CS: 6, CLK: 4 | SPI лінія для LED-матриці |
| **DS3231** | SDA: 8, SCL: 9 | I2C шина для модуля RTC |
| **DHT22** | DATA: 5 | Один провід для датчика DHT |
| **Battery** | GPIO 0 (ADC1_CH0) | Вхід дільника напруги (1:1) |

## Як розпочати

### 1. Збірка та прошивка
Для компіляції проекту вам знадобиться [ESP-IDF](https://github.com/espressif/esp-idf).

```bash
idf.py build
idf.py -p /dev/ttyACM0 flash monitor
```

### 2. Оновлення через OTA
Після першого прошити по кабелю, подальші оновлення можна робити через браузер за адресою `http://clock.local/ota.html` (логін/пароль вказані в `WebServerManager.h`).

## Структура проекту

- `main/main.cpp`: Точка входу, ініціалізація та запуск задач.
- `main/ClockManager.cpp`: Логіка рендерингу та циклів відображення.
- `main/BatteryMonitor.cpp`: Асинхронний моніторинг батареї з фільтрацією.
- `main/WebServerManager.cpp`: Веб-сервер та обробка OTA оновлень.
- `main/Max7219.cpp`: Драйвер LED-матриці з підтримкою повороту блоків.
- `main/RtcManager.cpp`: Робота з годинником реального часу DS3231.
- `main/DhtManager.cpp`: Зчитування даних з датчика DHT22.
- `main/SunTimeManager.cpp`: Розрахунок сходу/заходу сонця та керування нічним режимом.
- `main/MqttManager.cpp`: Передача даних у Home Assistant.
- `main/LogManager.cpp`: Збір та кільцевий буфер логів для API.
