# Smart Clock (ESP32-C3 + MAX7219 + DS3231 + DHT22)

Розумний годинник на базі ESP32-C3 з відображенням часу, температури та вологості на LED-матриці 8x32.

## Основні можливості

- **Циклічне відображення**: 5 секунд показує час (HH:MM), 2 секунди — температуру та вологість.
- **Синхронізація часу**: 
    - Пріоритетно через Wi-Fi (NTP).
    - Автономно через модуль RTC (DS3231) з механізмом повторних спроб при збої зчитування.
    - Ручна синхронізація через USB/Serial за допомогою Python-скрипта.
    - ESP-NOW синхронізація (маякування часу іншим пристроям).
- **Сенсори**: Вимірювання температури та вологості за допомогою датчика DHT22 (AM2302).
- **MQTT & Home Assistant**: 
    - Автоматичне виявлення пристрою (Discovery).
    - Передача температури та вологості кожну хвилину.
    - Округлення значень до 1 знака після коми.
- **Дизайн**: Кастомний шрифт 5x8 (на всю висоту матриці 8 пікселів) та система автоматичного центрування.

## Схема підключення (GPIO)

| Компонент | Пін | Опис |
|-----------|-----|------|
| **MAX7219** | DIN: 7, CS: 6, CLK: 4 | SPI лінія для LED-матриці |
| **DS3231** | SDA: 8, SCL: 9 | I2C шина для модуля RTC |
| **DHT22** | DATA: 5 | Один провід для датчика DHT |

## Як розпочати

### 1. Збірка та прошивка
Для компіляції проекту вам знадобиться [ESP-IDF](https://github.com/espressif/esp-idf).

```bash
idf.py build
idf.py -p /dev/ttyACM0 flash monitor
```

### 2. Синхронізація часу через Serial
Якщо Wi-Fi недоступний, можна синхронізувати годинник з часом вашого комп'ютера:

```bash
# Встановіть залежність pyserial
pip install pyserial
# Запустіть скрипт (порт автоматично визначено як /dev/ttyACM0)
python3 sync.py
```

## Структура проекту

- `main/main.cpp`: Точка входу, ініціалізація та запуск задач.
- `main/ClockManager.cpp`: Логіка рендерингу та циклів відображення.
- `main/Max7219.cpp`: Драйвер LED-матриці з підтримкою повороту блоків.
- `main/RtcManager.cpp`: Робота з годинником реального часу DS3231.
- `main/DhtManager.cpp`: Зчитування даних з датчика DHT22.
- `main/SunTimeManager.cpp`: Розрахунок сходу/заходу сонця та керування нічним режимом.
- `main/EspNowTimeSync.cpp`: Система розподіленої синхронізації через протокол ESP-NOW.
- `main/MqttManager.cpp`: Передача даних у Home Assistant та робота з MQTT.
- `sync.py`: Скрипт на Python для швидкої синхронізації часу через USB.
